Sentences Builder

If you know English but not very well, this service help you to build English sentences.

Just enter English text in left input box, and see on translating to native language in right box.

Suggestions of English words will be appear below (select if need using arrow Key and press space Bar).

Forgot some word? Type it on native language, variants of translating will be appear below, and you can select one of them.

Guess: if translating looks good then original text is also correct.
